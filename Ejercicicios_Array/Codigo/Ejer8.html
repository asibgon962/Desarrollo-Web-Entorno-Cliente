Función listar(arr)

Esta función recibe un array de cadenas y devuelve una sola cadena con los elementos separados por guiones.

Aquí aparecerá el resultado.

Código de la función:

function listar(arr) {
  var resultado = '';

  for (var i = 0; i < arr.length; i++) {
    resultado = resultado + arr[i];
    if (i < arr.length - 1) {
      resultado = resultado + '-';
    }
  }

  return resultado;
}
