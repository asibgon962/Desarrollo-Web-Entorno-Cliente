Función existeEn(letra, arr)

Esta función determina si una letra se encuentra dentro de un array de letras, sin usar métodos del objeto Array.

Aquí aparecerá el resultado.

Código de la función:

function existeEn(letra, arr) {
  for (var i = 0; i < arr.length; i++) {
    if (arr[i] === letra) {
      return true;
    }
  }
  return false;
}
