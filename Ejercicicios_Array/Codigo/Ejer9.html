Función servicio(día)

Esta función devuelve el nombre de la persona encargada del servicio en un día determinado. Si el día no existe, responde que no hay servicio.

Aquí aparecerá el resultado.

Turnos semanales:

  • lunes → María
  • martes → Luis
  • miércoles → Antonia
  • jueves → Pedro
  • viernes → Marisa

Código de la función:

function servicio(dia) {
  var turnos = [
    ['lunes', 'María'],
    ['martes', 'Luis'],
    ['miércoles', 'Antonia'],
    ['jueves', 'Pedro'],
    ['viernes', 'Marisa']
  ];

  var d = dia.toLowerCase();

  for (var i = 0; i < turnos.length; i++) {
    if (turnos[i][0] === d) {
      return 'Este día se encarga ' + turnos[i][1];
    }
  }

  return 'No hay servicio ese día';
}
