Función esVocal(letra)

Esta función devuelve true si la letra es una vocal (a, e, i, o, u), o false si no lo es. Funciona igual con mayúsculas y minúsculas.

Aquí aparecerá el resultado.

Código de la función:

function esVocal(letra) {
  var vocales = ['a', 'e', 'i', 'o', 'u'];
  var l = letra.toLowerCase();

  for (var i = 0; i < vocales.length; i++) {
    if (l === vocales[i]) {
      return true;
    }
  }
  return false;
}
